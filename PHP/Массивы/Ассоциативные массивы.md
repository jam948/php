\- массивы, использующие именованные ключи, которые вы им назначаете.

Есть два способа создания ассоциативного массива:

```php
$langs = array("Python" => 1991, "PHP" => 1995, "Go" => 2009);
```

или

```php
$langs["Python"] = 1991;
$langs["PHP"] = 1995;
$langs["Go"] = 2009;
```

```php
$langs = array("Python" => 1991, "PHP" => 1995, "Go" => 2009);
echo $langs["PHP"]; // выведет: 1995
```
