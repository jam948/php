[[Массивы]]

```php
$cities = array("Лондон", "Осло", "Вена", "Дели");
```
В качестве альтернативы вы можете назначить индекс вручную:

```php
$cities[0] = "Лондон";
$cities[1] = "Осло";
$cities[2] = "Вена";
$cities[3] = "Дели";
```

или так:
```php
```php
$arr = []; // пустой
$cities = ["Лондон", "Осло", "Вена", "Дели"];
```
В одном массиве можно объединять целые числа, строки и другие типы данных
