[[ООП]]

```php
<?php
interface Animal {
    public function voice();
}

class Cat implements Animal {
    public function voice() {
        echo "Мяу!";
    }
}

class Cow implements Animal {
    public function voice() {
        echo "Му!";
    }
}

$my_cat = new Cat();
$my_cat->voice(); // выведет: Мяу!

$my_cow = new Cow();
$my_cow->voice(); // выведет: Му!
```

Класс может реализовать несколько интерфейсов. Несколько интерфейсов можно указать, разделив их запятыми:

```php
class Cobra implements Animal, Reptile, Snake {
  // методы объявленые интерфейсами должны быть определены здесь
}
```
