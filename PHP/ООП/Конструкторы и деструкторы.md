### конструкторы
```php
class Cat {
    public $name;
    public $color;
    public function __construct($name, $color) {
        $this->name = $name;
        $this->color = $color;
    }
}
$barsik = new Cat("Барсик", "Белый");
```

### деструкторы 

```php
class Cat {
    public function __destruct() {
        echo "Объект уничтожен или скрипт завершен";
    }
}
$barsik = new Cat();
```
