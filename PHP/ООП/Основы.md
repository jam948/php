### Классы
class SimpleClass
{

    public $var = "Свойство";

    // объявление метода
    public function sayHello() {
        echo "Привет!";
    }

### Объекты
Для создания объекта класса используется ключевое слово `new`:

```php
$barsik = new Cat();
```

Для получения доступа к свойствам и методам объекта используйте конструкцию со стрелкой `**->**`. Обратите внимание, что при доступе к свойству не нужно указывать знак `$`, как в случае с обычной переменной:

```php
echo $barsik->name;
```

#### Пример
Давайте определим класс **Cat**, создадим объект класса, выполним присваивание и вызовем метод `voice()`:

```php
class Cat {
    public $name;
    public function voice() {
        echo "Мяу!";
  }
}
$barsik = new Cat();      // создание объекта класса
$barsik->name = "Барсик"; // присваивание имени
echo $barsik->name;       // выведет свойство: Барсик
$barsik->voice();         // выполнится метод и выведет: Мяу
```


### $this

```php
class Cat {
    public $name;
    public $color = "Белый";
    public function viewColor() {
        echo $this->color;
    }
}
$barsik = new Cat();
$barsik->viewColor(); // выведет: Белый

$murka = new Cat();
$murka->color = "Черный";
$murka->viewColor();  // выведет: Черный
```


### Область видимости классов
Доступ к свойствам и методам класса, объявленным как `public` (общедоступный), разрешён отовсюду. Модификатор `protected` (защищённый) разрешает доступ самому классу, наследующим его классам и родительским классам. Модификатор `private` (закрытый) ограничивает область видимости так, что только класс, где объявлен сам элемент, имеет к нему доступ.


### static 
```php
<?php
class Car {
    public static $engine = "ДВС";
    public static $wheels = 4;
    public static function engineType() {
        // ...
    }
}

echo Car::$wheels; // выведет: 4
```

Для доступа к статическому свойству из статического метода используется ключевое слово `self`

Например:

```php
<?php
class Car {
    public static $engine = "ДВС";
    public static $wheels = 4;
    public static function engineType() {
        echo self::$engine;
    }
}

Car::engineType(); // выведет: ДВС
```
