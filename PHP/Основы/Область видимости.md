[[Основы]]
```php
<?php
  $city = "Лондон";
  function get_city() {
    echo $city;
  }
  get_city();

// Error: Undefined variable: city
// Ошибка: Неопределенная переменная: city
```

Этот скрипт вызовет ошибку, так как переменная `$city` находится в глобальной области видимости и недоступна в функции `get_city()`.


#### ___global

```php
<?php
  $city = "Лондон";
  function get_city() {
    global $city;
    echo $city;
  }
  get_city();

// выведет: Лондон

```
```php
<?php 
$num1 = 42;
function  my_func() {
  global $num1;
  $num1 = 11;
}
my_func();
echo $num1;
?>
// выведет 11
```
