#### Константа 
- define(constant_name, value)
```php

```


### Float
```php

```

### String
Конкантенация - 
```php
$str1 = "Приветствую, ";
  $str2 = "Степик!";
  echo $str1.$str2;
```

Чтобы управляющие символы работали нужно записывать строку в двойные кавычки, а не одинарные

### Boolean



### Приведение типов
```php
<?php
  $int = 6;

```
