[[Работа с файлами]]

### fopen
$my_file = fopen(имя_файла, режим_открытия);
Существуют такие режимы для открытия файла (наиболее часто используемые, список не полный):  
**r**: открывает файл только для чтения  
**r+**: открывает файл для чтения/записи. Запись начинает с начала файла  
**w**: открывает файл только для записи. Стирает содержимое файла или создает новый файл, если он не существует. Запись добавляет с начала файла. Если в файле есть информация, она будет стерта  
**w+**: открывает файл для чтения/записи. Стирает содержимое файла или создает новый файл, если он не существует. Запись добавляет с начала файла. Если в файле есть информация, она будет стерта  
**a**: открывает файл только для записи. Создает новый файл, если он не существует. Запись добавляет в конце файла  
**a+**: открывает файл для чтения/записи. Создает новый файл, если он не существует. Запись добавляет в конце файла

### fwrite
fwrite(имя_файла, данные);

### fclose 
Функция `fclose()` закрывает открытый ранее с помощью функции `fopen()` файл и возвращает **true** в случае успеха или **false** в случае неудачи.

### Запись нескольких данных
Делается с помощью массива и цикла foreach
`$capitals = ["Лондон", "Пекин", "Вашингтон", "Осло", "Токио"];`

`$my_file = fopen("capitals.txt", "w");`

`foreach ($capitals as $cap) {`

    `fwrite($my_file, $cap);`

`}`

`fclose($my_file);`
