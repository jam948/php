Функция `file()` читает содержимое файла и помещает его в массив. Каждый элемент массива соответствует строке файла, с символами новой строки включительно.

Пример использования:

```php
$lines = file("capitals.txt");
foreach ($lines as $line) {
  echo $line;
}
```

В этом примере построчно будет выведен весь текст из файла `capitals.txt`
